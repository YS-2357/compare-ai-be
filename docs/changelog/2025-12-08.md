# 변경 이력 - 2025-12-08

## 주요 변경 사항
- FastAPI 제목/설명 최신화 (`app/config.py`, `app/main.py`)
  - 제목을 `Compare-AI Backend API`로 통일하고 `strip('"')` 처리로 잘못된 따옴표 제거.
  - Description에 모델 오버라이드, NDJSON 스트리밍, 사용량 헤더, 관리자 bypass 반영.
- `/api/ask` 문서화 강화 (`app/api/routes.py`)
  - summary를 “LangGraph 스트리밍 질의 (NDJSON)”로 변경.
  - 200 응답에 partial 예시(JSON) 추가, 401/503 응답 설명 추가.
- `/usage` 문서화 강화 (`app/rate_limit/router.py`)
  - response_model 지정 및 예시(JSON) 추가, 401/503 응답 설명/summary 추가.
- 스키마 예시·설명 추가 (`app/api/schemas/ask.py`, `app/api/schemas/auth.py`, `app/api/schemas/common.py`)
  - AskRequest의 question/turn/max_turns/history/models 필드에 description+examples 추가.
  - Login/Register 요청·응답 필드에 description+examples 추가(토큰, 이메일, 패스워드, 만료 등).
  - UsageResponse limit/remaining/bypass에 examples 추가.
